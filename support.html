Français

Signaler un problème

Indiquez la version de l’application, la version iOS/iPadOS ou Android, le modèle d’appareil, l’écran concerné, les actions précédentes, le résultat observé et le résultat attendu. Une capture expurgée de toute donnée sensible peut être jointe si nécessaire.

Achats iPhone et iPad

Le téléchargement est gratuit. L’essai Premium unique dure sept jours, ne se renouvelle pas et ne déclenche aucun prélèvement automatique. À son expiration, les fonctions Premium nécessitent le déblocage permanent distinct affiché par l’App Store.

Pour restaurer l’essai reconnu ou le déblocage permanent, ouvrez Plus → Accès Premium → Restaurer les achats avec le même compte Apple. Pour un code d’offre Apple, utilisez l’action de validation de code App Store proposée dans le même écran. Ne communiquez jamais de reçu, mot de passe, code 2FA ou information bancaire par e-mail.

Achats Android

Utilisez l’action de restauration Google Play avec le même compte Google que celui ayant effectué l’achat. La disponibilité et les conditions des offres Android sont celles affichées par Google Play.

Données locales et suppression

Les favoris, notes, PN/IPC personnels, logbook, documents, photos, rapports et exports sont principalement conservés sur l’appareil. Utilisez les commandes de suppression ou de réinitialisation de l’application. Les fichiers exportés doivent être supprimés séparément dans leur emplacement externe. Une désinstallation peut supprimer les données locales ; sauvegardez volontairement vos contenus personnels avant toute suppression si l’application propose un export approprié.

English

Report an issue

Include the app version, iOS/iPadOS or Android version, device model, affected screen, preceding actions, observed result and expected result. A screenshot stripped of sensitive information may be attached when needed.

iPhone and iPad purchases

The download is free. The one-time Premium trial lasts seven days, does not renew and causes no automatic charge. When it expires, Premium features require the separate permanent unlock displayed by the App Store.

To restore a recognized trial or permanent unlock, open More → Premium Access → Restore purchases using the same Apple Account. To redeem an Apple offer code, use the App Store code-redemption action on the same screen. Never email a receipt, password, 2FA code or payment information.

Android purchases

Use the Google Play restore action with the same Google Account used for the purchase. Android offer availability and terms are those displayed by Google Play.

Local data and deletion

Favorites, notes, personal PN/IPC entries, logbook, documents, photos, reports and exports are primarily stored on the device. Use the app’s delete or reset controls. Exported files must be removed separately from their external location. Uninstalling may remove local data; deliberately back up personal content before deletion when an appropriate export is available.
